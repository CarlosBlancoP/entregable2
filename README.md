# Proyecto de Análisis de Datos en Python

## 📌 Descripción
Este proyecto consiste en construir una aplicación de análisis de datos en Python, que incluye:
- Funciones para cargar y limpiar datos.
- Limpieza de valores nulos y textos.
- Operaciones de merge, filtrado y agrupación.
- Responder preguntas clave de análisis.
